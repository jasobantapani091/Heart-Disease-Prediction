# ❤️ Heart Disease Prediction using Machine Learning

## 📌 Project Overview
Heart disease is one of the leading causes of death worldwide.  
This project aims to predict the **presence of heart disease** in a patient using **machine learning classification algorithms** based on medical attributes.

The system analyzes health parameters such as age, blood pressure, cholesterol, and heart rate to provide accurate predictions.

---

## 🎯 Objectives
- To predict whether a person has heart disease
- To analyze the most important medical factors affecting heart disease
- To compare different machine learning classification models
- To improve early diagnosis using data-driven techniques

---

## 📊 Dataset Information
- **Dataset Name:** Heart Disease Dataset
- **Source:** UCI Machine Learning Repository
- **Number of Records:** 303
- **Number of Features:** 14
- **Target Variable:**  
  - `0` → No Heart Disease  
  - `1` → Heart Disease Present  

---

## 🧬 Dataset Features
| Feature | Description |
|------|------------|
| age | Age of the patient |
| sex | Gender (1 = male, 0 = female) |
| cp | Chest pain type |
| trestbps | Resting blood pressure |
| chol | Serum cholesterol |
| fbs | Fasting blood sugar |
| restecg | Resting ECG results |
| thalach | Maximum heart rate achieved |
| exang | Exercise-induced angina |
| oldpeak | ST depression |
| slope | Slope of peak exercise ST segment |
| ca | Major vessels colored |
| thal | Thalassemia |
| target | Heart disease (0 or 1) |
